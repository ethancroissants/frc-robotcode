Quickstart

From sealed-box Acuity to robot code reading AprilTag poses in about ten minutes.

  1. Plug Acuity into the RoboRIO's USB-A port. Power on.
  2. On a phone, connect to the open WiFi Acuity-Setup-XXXX. The captive portal will pop a setup form (or browse to http://192.168.50.1/ if it doesn't).
  3. Fill in your team number, robot WiFi SSID, and password. Save. The device reboots and joins the team radio at 10.TE.AM.11.
  4. Open this Manager — your device shows up in the Devices tab automatically (mDNS discovery).
  5. Open the Libraries tab, pick your team's language, and point the picker at your robot project folder. Manager drops the vendordep JSON or pip dep into place.
  6. Add 3 lines to your robot code (see your language's section below) and you're reading AprilTag data.

How it fits together

Acuity is a Pi Zero 2 W running a vision pipeline that publishes AprilTag detections to NetworkTables 4. Three pieces:

The device
Camera → OpenCV capture → pyapriltags → /acuity/* NT4 topics. Also serves a web dashboard at :8080 (live feed, settings, network controls). Mounts via USB-A on the RoboRIO.
The robot library (Java / Python / C++)
Tiny wrapper around NT4 that handles connection state, staleness, JSON parsing of the all-tags array, and gives you a typed API (getBestTag() → Optional<TagDetection> with distanceMeters / yawDeg / pitchDeg fields).
This Manager
Driver-station companion. Discovers devices, one-clicks firmware updates, opens an in-app SSH terminal, installs the robot library into your project, and embeds the camera feed (Camera tab).

All three speak NT4 over the team radio. The robot never opens an HTTP socket to Acuity — it reads NT4 topics like every other component.

Java (WPILib)

Install

The Libraries tab installs the vendordep automatically. For manual install:

  1. In VS Code, open your robot project.
  2. Click the WPILib palette icon → Manage Vendor Libraries → Install new library (online).
  3. Paste: 
    https://raw.githubusercontent.com/ethancroissants/frc-robotcode/master/acuity/libraries/Acuity.json
  4. OK to add. Rebuild the project.

Minimal usage

import tech.acuity.AcuityVision;
import tech.acuity.TagDetection;

public class Robot extends TimedRobot {
  private final AcuityVision vision = AcuityVision.getInstance();

  @Override
  public void robotPeriodic() {
    vision.getBestTag().ifPresent(tag -> {
      SmartDashboard.putNumber("acuity/id",         tag.id);
      SmartDashboard.putNumber("acuity/distance_m", tag.distanceMeters);
      SmartDashboard.putNumber("acuity/yaw_deg",    tag.yawDeg);
    });
  }
}

API reference

MethodReturnsNotes
AcuityVision.getInstance() AcuityVision Singleton — one client per process.
vision.getBestTag() Optional<TagDetection> Largest tag in frame; empty when no tag or device offline.
vision.getAllTags() List<TagDetection> Every tag in current frame.
vision.getBestObject() Optional<ObjectDetection> Object detector output (off by default).
vision.getHealth() DeviceHealth CPU %, SoC °C, uptime seconds.
vision.isConnected() boolean Heartbeat fresh within 250 ms.

TagDetection fields

FieldTypeMeaning
idintAprilTag ID. -1 means "no tag" (you'll never see this through Optional, but raw NT4 readers will).
distanceMetersdoubleDistance from camera to tag center.
yawDegdoubleCamera-to-tag yaw, signed. Negative = left of center.
pitchDegdoubleCamera-to-tag pitch, signed.
tx / tydoublePixel offset from frame center, normalized to [-1, 1] (resolution-independent).
areadoubleFraction of frame the tag occupies, 0..1.
timestampdoubleFPGA seconds when the source frame was captured.
decisionMargindoublepyapriltags confidence — bigger = stronger detection.

Common patterns

Auto-aim with the best tag

vision.getBestTag().ifPresent(tag -> {
  // Drive to face the tag (yaw_deg → heading PID setpoint).
  double headingErrorDeg = tag.yawDeg;
  double turnPower = -kP_HEADING * headingErrorDeg;
  drivetrain.arcadeDrive(0, turnPower);
});

Filter to specific tag IDs

Set the preferred-tag-ID list in the dashboard's Settings tab — Acuity then picks "best" only from those IDs. Or filter client-side: 

var speakerTag = vision.getAllTags().stream()
    .filter(t -> t.id == 7 || t.id == 4)   // 2025 speaker tags
    .max(Comparator.comparingDouble(t -> t.area));
speakerTag.ifPresent(tag -> { /* aim */ });

Handle the device dropping mid-match

getBestTag() returns empty whenever the heartbeat is stale (>250 ms). Check isConnected() if you want to distinguish "no tag in view" from "device unreachable": 

if (!vision.isConnected()) {
  // Device is gone — fall back to encoders + IMU only.
  setVisionTrustWeight(0.0);
} else {
  setVisionTrustWeight(1.0);
}

Minimal usage

import wpilib
import acuity_vision

class MyRobot(wpilib.TimedRobot):
    def robotInit(self):
        self.vision = acuity_vision.AcuityVision()

    def robotPeriodic(self):
        tag = self.vision.get_best_tag()
        if tag is not None:
            wpilib.SmartDashboard.putNumber("acuity/id",         tag.id)
            wpilib.SmartDashboard.putNumber("acuity/distance_m", tag.distance_m)
            wpilib.SmartDashboard.putNumber("acuity/yaw_deg",    tag.yaw_deg)

Common patterns

# Drop in commands/auto_aim.py
import commands2
import acuity_vision

class AutoAim(commands2.Command):
    def __init__(self, drivetrain):
        super().__init__()
        self.drive = drivetrain
        self.vision = acuity_vision.AcuityVision()
        self.addRequirements(drivetrain)

    def execute(self):
        tag = self.vision.get_best_tag()
        if tag is None:
            self.drive.arcadeDrive(0, 0)
            return
        turn = -0.04 * tag.yaw_deg
        self.drive.arcadeDrive(0, turn)

    def isFinished(self):
        tag = self.vision.get_best_tag()
        return tag is not None and abs(tag.yaw_deg) < 1.5

Versioning policy

  • Adding a field never bumps version.
  • Renaming or removing a field bumps /acuity/version. Robot libraries warn but degrade gracefully.
  • Restructuring the layout bumps version + ships a migration note.

REST endpoints (device)

The on-device dashboard exposes a small REST surface on port 8080. You normally don't touch it from robot code — Manager and the dashboard UI consume it. Useful for debugging.

EndpointReturns
GET /HTML dashboard.
GET /stream.mjpgmultipart MJPEG video.
GET /api/detectionsOne-shot JSON snapshot of current tags.
WS /api/detections.wsLive stream of detection snapshots, one per processed frame.
GET /api/healthJSON: build, FPS, CPU, temp, uptime, network state.
GET /api/settingsJSON of current camera + detection + NT4 settings.
POST /api/settingsUpdate settings. Body is JSON with any subset of the keys.
POST /api/rebootReboots the Pi after a 1 s grace.
POST /api/forget-wifiRemoves /boot/firmware/acuity.conf and reboots → AP-mode setup.
GET /api/logs?lines=NTail of the dashboard's systemd journal.

acuity.conf options

Per-Pi config on the FAT32 boot partition at /boot/firmware/acuity.conf. Editable from any laptop without booting the Pi. 

# Team number drives hostname (`acuity-NNNN.local`) and the
# FRC-standard static IP `10.TE.AM.11`.
TEAM=1279

# Robot WiFi SSID + password. Leave PSK blank for an open network.
SSID=1279
PSK=robotpassword

# Country code for WiFi regulatory domain.
COUNTRY=US

Delete this file and reboot to re-trigger the AP-mode setup wizard.

Troubleshooting

SymptomLikely causeFix
Manager shows no devices Laptop on the wrong WiFi, or device hasn't booted Confirm laptop is on the team radio. Wait 60 s after Pi boot. Hit Rescan.
Camera tab spinner never resolves Pi camera ribbon loose, or another process owns /dev/video0 Reseat the camera ribbon. SSH in (Terminal tab) and run fuser /dev/video0 to find a conflicting process.
Robot code reads id == -1 only Tag size mismatch — wrong tag_size_m in dashboard Settings, or no AprilTag in view Check the dashboard Live tab. If tags appear there but not in robot code, check NT team number under Settings.
isConnected() bouncing true/false Spotty WiFi between Pi and radio Open dashboard Network tab — check IP. Reboot the Pi via Manager's Reboot button.
SmartScreen blocks Manager on first run We ship unsigned (no $200/yr cert) Click More info → Run anyway. Same on every Manager update.
Forgot WiFi credentials — Forget WiFi via Manager (Devices → tile → "Forget WiFi") or pop the SD card and edit /boot/firmware/acuity.conf.
