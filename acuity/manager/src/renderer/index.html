Welcome to Acuity Manager

        

          This is the laptop side of the Acuity vision coprocessor.
          From here you'll set up new devices, install vision
          libraries into your robot project, and update firmware in
          one click.
        

        

          We'll walk you through it once. You can re-open this guide
          anytime from the ? button in the top right.
        

        

          
        

      


      
      

        
What are you doing today?

        

          
          
          
        

      


      
      

        
Connect to the device's setup network

        
    
          
  1. Power the Acuity device on. The status light pulses
              yellow while it boots — wait about 30 seconds.
    2. 
          
  2. On this laptop, open the WiFi menu and
              connect to the open network named
              Acuity-Setup-XXXX.
    3. 
          
  3. A setup form should pop up automatically. If not,
              open a browser and visit
              http://192.168.50.1/.
    4. 
          
  4. Fill in your team number, your robot's WiFi name and
              password, then save. The device reboots and joins the
              team radio.
    5. 
          
  5. Reconnect this laptop to your team WiFi, then come
              back here and click Find my device.
    6. 
        

        

          Tip: this same flow works for any Acuity device — you can
          re-trigger it later from the dashboard's
          Forget WiFi button.
        

        

          
          
        

      


      
      

        
Make sure you're on the team WiFi

        
    
          
  1. Connect this laptop to your team's robot WiFi.
    2. 
          
  2. Acuity advertises itself via mDNS — devices appear in
              the Devices tab automatically once they're on the
              network.
    3. 
        

        

          Not seeing it? Plug in a monitor + keyboard on the device
          and run ip addr show wlan0 to confirm it has
          a team-WiFi IP. Or: Forget WiFi and run through the
          captive-portal setup again.
        

        

          
          
        

      


      
      

        
Add Acuity to your robot project

        

          Pick the language your team uses. Manager will detect your
          robot project and add the library for you — no manual
          vendordep editing.
        

        

          
          
          
        

        

          
        

      


      
      

        
You're set

        

          The Devices tab shows everything Acuity finds on your
          network. Click a device for one-click update / reboot /
          terminal / forget-WiFi controls.
        

        

          The Libraries tab installs the vision client into your
          robot project. The Logs tab tails everything Manager
          does.
