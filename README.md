## ▶ Start here

**Just run `start.py`** — it opens a friendly control panel with buttons for
install, deploy, update, simulator, and docs. Everything else flows from there.

| Your computer | Double-click           | Or from a terminal     |
| ------------- | ---------------------- | ---------------------- |
| Windows       | `START.bat`            | `python start.py`      |
| macOS         | `start.command`        | `python3 start.py`     |
| Linux         | `start.command`        | `python3 start.py`     |

That's it. The control panel handles the rest.
